## 🚀 Quick Start

### Windows:
1. Download from: https://github.com/chibuenyim/UniversalDiskCleanupTool/releases
2. Extract ZIP
3. Double-click `START.bat` or run:
   ```powershell
   pwsh -File cleanup.ps1 --All
   ```

### macOS:
```bash
# Install PowerShell
brew install powershell

# Download and run
chmod +x start.sh
./start.sh
```

### Linux:
```bash
# Install PowerShell (Ubuntu/Debian)
sudo apt-get install -y powershell

# Download and run
chmod +x start.sh
./start.sh
```

That's it! The cleanup will start automatically.

## 🛡️ Is It Safe?

**YES!** We only clean:
- ✅ Temporary files
- ✅ Cache files
- ✅ Package downloads
- ✅ Build artifacts
- ✅ System logs

**NEVER touch:**
- ❌ Your documents
